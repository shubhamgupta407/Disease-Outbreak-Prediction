# Dengue Outbreak Prediction Dashboard

A machine learning–based system designed to predict dengue outbreak risk using climate and demographic data. The project provides an interactive dashboard that visualizes outbreak risk and predicted case counts for different regions.


## Overview

Dengue outbreaks are influenced by environmental and demographic factors such as temperature, rainfall, humidity, and population density.  
This project leverages machine learning models to analyze historical data and predict potential outbreak risks.

The system helps identify high-risk regions and enables better preparedness through data-driven insights.


## Features

- Dengue outbreak risk prediction
- Case count estimation
- Interactive dashboard visualization
- Climate and demographic data analysis
- Machine learning ensemble modeling

## Machine Learning Models

The prediction system uses ensemble learning techniques including:

- Random Forest
- XGBoost
- ExtraTrees

These models are trained on historical climate and demographic datasets to identify patterns associated with dengue outbreaks.
